Primera ley de Newton

Un objeto permanecerá en reposo o con movimiento rectilíneo uniforme (MRU) a menos que sobre él actúe una fuerza extrena.

Velocidad Media

El vector velocidad media se describe como la relación entre el cambio de posición del móvil (r2-r1) y el tiempo empleado (t2-t1).

V=(r2-r1)/(t2-t1)

Velocidad Instantánea

Para definir la velocidad en un punto debemos acotar el tiempo (tiempo tienda a 0), se describe como el límite (t->0) de la relación entre el vector desplazamiento (Δr) y el intervalo de tiempo (Δt).

V.int = lim(Δr/Δt) = dr/dt

Aceleración Media

El vector aceleración media se describe como la relación entre el cambio de velocidad (v2-v1) y el tiempo empleado (t2-t1).

a=(v2-v1)/(t2-t1)

Aceleración Instantánea

Para definir la aceleración en un punto debemos acotar el tiempo (tiempo tienda a 0), se describe como el límite (t->0) de la relación entre el vector velocidad (Δv) y el intervalo de tiempo (Δt).

a.inst = lim(Δv/Δt) = dv/dt

a.inst = d/dt (dr/dt) = d²r/dt²
